Data Deletion Instructions

Last updated: 2026-01-28

If you want to request deletion of data associated with the ClawdBot integration for @tommythebernese, follow the steps below.

Option 1 — Remove app access (recommended)

  1. Go to your Meta/Facebook settings.
  2. Open Business Integrations (or "Apps and Websites").
  3. Find ClawdBot and remove access.

Removing access stops any further data processing and disconnects the integration.

Option 2 — Request deletion by email

Email us at: tommythebernese@gmail.com with:

  • Subject: Data Deletion Request
  • Your Instagram username
  • A brief description of what you want deleted (e.g., logs, stored message metadata)

What we delete

Upon a valid request, we will delete stored data we control, which may include:

  • Stored webhook event logs/metadata (if any)
  • Stored automation state (e.g., deduplication records)

Note: We cannot delete data that Meta controls on their platforms; removing app access is the correct method for that.

Timing

We typically process deletion requests within 7–30 days.
